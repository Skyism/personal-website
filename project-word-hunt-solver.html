Word Hunt Solver

Over the course of summer, I created a GamePigeon Word Hunt Solver using Python. I implemented a depth-first search (DFS) algorithm to efficiently traverse a customizable grid and identify valid words and developed a dynamic board generation feature that allows users to input a string to set up various game boards.
