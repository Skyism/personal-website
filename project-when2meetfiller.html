When2Meet Autofiller

During the fall semester, I developed a program that automates the process of filling out availability on When2Meet, a scheduling platform. The tool integrates with the Google Calendar API to extract the user's schedule and uses Selenium to input the availability directly into the When2Meet website. This streamlines the scheduling process, saving users time by eliminating the need for manual entry.
